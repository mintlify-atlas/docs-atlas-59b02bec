---
title: "Firmware Settings"
description: "Customise the filename template used when saving macOS IPSW firmware files."
---

Access these settings from **Mist &gt; Settings &gt; Firmwares**.

## Overview

macOS Firmwares are **IPSW** files used to restore [Apple Silicon Macs](https://support.apple.com/en-us/HT211814) via [Apple Configurator](https://apps.apple.com/us/app/apple-configurator/id1037126344?mt=12).

<Note>
  Firmware metadata is sourced from the [IPSW Downloads API](https://ipswdownloads.docs.apiary.io/). Mist queries this API to populate the Firmwares list.
</Note>

## Firmware Filename

The **Firmware filename** field controls the name given to each downloaded IPSW file. It supports dynamic template variables that are substituted with real values at download time.

### Template variables

| Variable | Replaced with | Example |
|---|---|---|
| `%NAME%` | The macOS release name | `macOS Tahoe` |
| `%VERSION%` | The macOS version number | `26.0` |
| `%BUILD%` | The build identifier | `25A354` |

### Default template

```
Install %NAME% %VERSION%_%BUILD%.ipsw
```

### Example output

With the default template and a macOS Tahoe 26.0 firmware:

```
Install macOS Tahoe 26.0_25A354.ipsw
```

The preview below the filename field updates in real time as you type, showing you exactly what the output filename will look like before you download anything.

## Reset to Defaults

Clicking **Reset to Defaults** restores the firmware filename to:

```
Install %NAME% %VERSION%_%BUILD%.ipsw
```
