## What Mist does

Mist connects to Apple's servers to retrieve a complete, up-to-date catalog of available macOS versions. You can browse, filter, and download exactly what you need — with checksum validation to ensure every download is intact.

## Key features

- **Apple Silicon support** — Download IPSW Firmware files with SHA-1 checksum validation
- **Intel Mac support** — Build .app, .dmg, .iso, and .pkg installers from macOS Catalina through the latest release
- **Legacy macOS** — Supports Mac OS X Lion 10.7 through macOS Sierra 10.12 on Intel Macs
- **Codesigning** — Optionally codesign disk images and packages with your Developer ID
- **Caching** — Cache downloads locally to speed up repeated build operations
- **List export** — Export the full firmware/installer catalog as CSV, JSON, Plist, or YAML
- **Automatic retries** — Configurable retry count and delay for resilient downloads
